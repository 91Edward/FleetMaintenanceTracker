# Fleet Maintenance Tracker

A C# console application for managing a fleet of trucks and tracking maintenance information.

## Features

- Add trucks to fleet
- Track service dates
- Calculate next service due (6 weeks)
- Log vehicle defects
- Track truck mileage
- Fleet summary dashboard

## Technologies Used

- C#
- .NET Console Application
- Object-Oriented Programming

## Example Use Case

This project simulates a logistics fleet management system. It was inspired by real-world fleet operations and vehicle maintenance tracking.

## Future Improvements

- Save fleet data to file or database
- Search trucks by registration
- Update truck mileage
- Add maintenance alerts
- Build a graphical interface
